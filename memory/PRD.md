# PRD - Jingping e Hijo Landing Page

## Problem Statement
Landing Page moderna y profesional para "Jingping e Hijo" - Café de Especialidad, Matcha Bar & Brunch ubicado en Madrid. Fusión cultural asiática-española con estética Japandi/Minimalista.

## User Personas
- **Amantes del café**: Buscan café de especialidad y experiencias únicas
- **Nómadas digitales**: Necesitan un espacio de trabajo tranquilo
- **Entusiastas del brunch**: Buscan brunch de fusión de calidad
- **Público wellness**: Interesados en matcha ceremonial

## Core Requirements (Static)
- Landing page 100% responsive (Mobile First)
- Sistema de reservas con selección de fecha/hora
- Paleta Japandi: Verde Matcha (#B8C480), Crema, Terracota
- Secciones: Hero, About, Menu, Gallery, Footer
- Integración Google Maps
- Links a redes sociales (Instagram)

## What's Been Implemented (Feb 2026)

- Frontend: React + Tailwind CSS + Shadcn/UI
- Backend: FastAPI + MongoDB
- Fuentes: Playfair Display + Lato
